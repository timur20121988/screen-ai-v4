# 🤖 ScreenAI

**[English]**  
ScreenAI is a lightweight, system tray utility that instantly analyzes screenshots using GPT-4o. Capture regions or the full screen, get instant answers copied to your clipboard, and keep a history of your queries. Perfect for quick explanations, coding help, or solving tests directly from your screen.

**Key Features:**
- 📸 **Instant Capture:** Screenshot full screen (Alt+Shift+A) or select a region (F10).
- 🧠 **Smart AI:** Powered by GPT-4o and GPT-4o Mini (switchable).
- ⚡ **Auto-Copy:** AI responses are automatically copied to your clipboard.
- 📝 **History:** View your last 20 queries and answers anytime.
- 🚀 **Autostart:** Toggle launch on Windows startup.
- 🛡️ **Optimized:** Lazy loading, caching, and minimal resource usage.

**[Русский]**  
ScreenAI — это легкая утилита для системного трея, которая мгновенно анализирует скриншоты с помощью GPT-4o. Выделяйте области или снимайте весь экран, получайте ответы, автоматически скопированные в буфер обмена, и просматривайте историю запросов. Идеально подходит для быстрых объяснений, помощи в коде или решения тестов прямо с экрана.

**Главные возможности:**
- 📸 **Мгновенный снимок:** Весь экран (Alt+Shift+A) или область (F10).
- 🧠 **Умный ИИ:** Работает на GPT-4o и GPT-4o Mini (можно переключать).
- ⚡ **Авто-копирование:** Ответ ИИ сразу попадает в буфер обмена.
- 📝 **История:** Просмотр последних 20 запросов и ответов в любое время.
- 🚀 **Автозапуск:** Возможность включить запуск при старте Windows.
- 🛡️ **Оптимизация:** Ленивая загрузка, кэширование и минимальное потребление ресурсов.

## 🛠 Usage / Использование

1. Add your `OPENAI_API_KEY` to the `.env` file or set it in your environment variables.
2. Run `ScreenAI.exe`.
3. Use hotkeys or the tray menu to start capturing!
